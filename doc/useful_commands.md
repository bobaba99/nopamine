For frontend initialization `npm create vite@latest frontend -- --template react-ts`

For pip `pip3` or `python -m pip`

For supabase upload to local server `supabase db reset`

For create new supabase sql script `supabase migration new [SCRIPT_NAME]`

Run backend `uvicorn main:app --reload`

Run tests `python3 -m pip install pytest python-dotenv google-genai`

when deploying to render, VITE_API_URL should be backend url on Render
